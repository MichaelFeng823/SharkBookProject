# SharkBookProject

## Qt C++ 安卓开发 高仿鲨鱼记账APP
 
#### 开发环境

- Qt5.14.2
- 构建套件：
    - Android for armeabi-v7a arm64-v8a x86 x86_64(clang Qt 5.14.2 for Android)
    - Jdk1.8.0_92
    - Android-SDK25.2.5
    - Andioid-NDK21.4
- 逍遥模拟器（开发调试使用）
